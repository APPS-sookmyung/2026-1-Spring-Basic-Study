# Spring basic study 2주차

## [ 섹션 3 ]
**1. 웹(Web)이란?**   
: 인터넷 (Internetional Network)이라는, 전세계가 연결되어 있는 신경망 위에서 정보를 공유하는 공간

- 웹의 시초 ( [info.cern.ch](https://info.cern.ch) )   
: 웹이 만들어진 근본적인 목적은, 전세계 사람들이 쉽게 정보를 공유하고 접근하기 위함이다.

- HTML (Hyper Text Markup Language)   
: 링크를 걸어줄 수 있는 (하이퍼 텍스트) 문서의 구조를 나타내는 언어 (마크업 랭귀지)

**2. HTTP (Hyper Text Transfer Protocol)**   
: 웹 환경에서 클라이언트(브라우저)와 서버가 서로 통신하기 위해 따르는 약속

- HTTP의 역할 : 인터넷 상에서 데이터를 요청하고 응답받는 통신 프로토콜 (= 약속, 규약)

- 받는 사람, 목적, 내용, 보내는 사람을 포함한다. (마치 편지를 쓸 때의 양식 처럼)
