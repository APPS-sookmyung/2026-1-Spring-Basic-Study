#### 첫 웹페이지 [first web](https://info.cern.ch/)
- 정보만을 공유하기 위해 만들어진 것이 첫 웹페이지
- 전 세계 사람들이 쉽게 정보를 공유하고 접근하기 위한 것이 웹페이지!
- WWW

#### HTML
: Hyper Text Markup Language
- 텍스트에 초능력(링크 기능)을 부여하고 문서의 구조(제목, 본문 등)를 정의하는데 사용되는 웹의 기본 마크업 언어
- 태그를 통해 초능력을 부여하고 문서의 구조를 정의 가능하다.

#### 웹(web)과 인터넷 (Internet)
- 웹은 인터넷상의 공간
- International Network(인터넷)은 전 세계가 연결된 신경망
- 웹은 인터넷이라는 거대한 통신 네트워크 위에서 정보를 공유하는 공간


#### HTTP
: Hyper Text Transfer Protocol
- 인터넷 세상을 돌아다니기 위해 필요한 **약속**
- 프로토콜 = 약속/규약
- 웹 환경에서 클라이언트(브라우저)와 서버가 서로 통신하기 위해 따르는 약속 또는 규약
- 인터넷 상에서 데이터를 요청하고 응답받는 통신 프로토콜
