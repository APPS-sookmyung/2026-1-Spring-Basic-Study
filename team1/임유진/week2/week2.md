## 웹이란?

- **[웹의 시작](https://info.cern.ch/)**
  - 정보를 공유하기 위해 탄생
  - 인터넷을 통해 연결된 전 세계의 컴퓨터에서 하이퍼텍스트(링크)를 기반으로 정보를 공유하는 거대한 정보 공유 시스템이자 온라인 공간
  - **WWW (World Wide Web)**: 웹의 정식 명칭, 줄여서 Web 또는 W3라고도 불림

- HTML (Hyper Text Markup Language)
  - 웹 페이지의 구조와 내용을 정의하는 표준 마크업 언어
  - 태그(<>)를 사용하여 콘텐츠를 구조화하고 링크를 연결하며, 웹 브라우저가 내용을 해석해 표시하도록 도움

---

## 인터넷이란?

- **인터넷**
  - 네트워크를 하나로 묶는 기술
  - TCP/IP 프로토콜을 기반으로 전 세계의 컴퓨터와 네트워크를 유,무선 기술로 연결한 거대 글로벌 통신망
    TCP/IP : 서로 다른 기종의 컴퓨터들이 인터넷상에서 데이터를 안정적으로 주고받기 위한 통신 프로토콜의 집합

- **HTTP (Hyper Text Transfer Protocol)**
  - 웹 브라우저와 서버가 HTML, 이미지 등 하이퍼미디어 문서를 주고 받기 위해 사용하는 핵심 애플리케이션 계층 통신 규약

> 웹은 인터넷이라는 거대한 통신 네트워크 위에서 정보를 공유하는 공간
