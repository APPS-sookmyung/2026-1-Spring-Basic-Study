### 스프링 부트

- 스프링은 **자바**로 구현된 프레임워크
- **웹 개발**이 목표!

### WEB

- 웹의 시작 : [info.cern.ch](http://info.cern.ch)
- 웹은 정보를 공유하기 위해 나타남
- **HTML** : hyper text markup language
    - 초능력이 있는 문자로 구조를 나타내는 언어
- 우리 **모두**를 위한 **정보 공유**의 공간

### 인터넷 vs 웹

- 웹은 인터넷 상의 공간
- International Network - 인터넷
    - 국제적으로 연결되어 있는 네트워크
    - 전 세계가 **연결**되어 있는 신경망
- 인터넷의 어느 한 지점에 공간을 만드는 것이 웹
    - 전 세계에 있는 사람이 해당 공간에 접근 가능

### HTTP

- 인터넷 세상을 돌아다니기 위해 필요한 **약속**
- Hyper Text Transfer Protocol
    - 프로토콜 = 약속, 규약
- 예시 : 편지 - 받는 사람, 목적, 내용, 보내는 사람
- 네이버라는 공간에 가고 싶다면
    - http://naver.com
